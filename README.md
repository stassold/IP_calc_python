# IP Address Calculator
A simple application for calculating basic parameters of an IP address and subnet mask.

# Requirements
To run the application, you need to have Python 3 and the PyQt5 library installed.

You can install the PyQt5 library using pip:

pip install pyqt5

# Usage
Launch the application by running python ip_calculator.py.
Enter the IP address in the format x.x.x.x.
Select the subnet mask from the dropdown list.
Click the "Calculate" button.
The result will be displayed in the application window.

# Calculated Parameters
The application calculates the following parameters of an IP address and subnet mask:

Network
Broadcast address
Subnet mask
Number of hosts
IP address in binary format
